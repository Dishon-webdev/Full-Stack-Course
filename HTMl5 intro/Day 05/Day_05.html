This is a simple paragraph demonstrating how you can style HTML using CSS. Depending on your project, you can apply styles inline, internally, or externally to achieve different effects and levels of maintainability.


Inline CSS

Inline styles are added directly to HTML elements. They’re quick for testing or small changes but can become messy in large projects.

Internal CSS

Internal styles are written within a "style" tag inside the HTML document's "head". They are useful for single-page websites or when styles don’t need to be reused across multiple pages.

External CSS

External stylesheets keep your design separate from content, making your code cleaner and easier to maintain across multiple pages.
